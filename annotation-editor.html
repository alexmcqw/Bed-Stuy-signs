Annotation Position Editor

How to use:

1. Click an annotation to select it (green border)

2. Drag the annotation label to change its position (both horizontal and vertical)

3. Click on the image to set where the arrow points (targetX, targetY) - works when an annotation is selected

4. Use the controls below to edit text, side, and positions manually

5. Click "Generate Code" to create the updated JavaScript code

6. Copy the code and replace the initAnnotations() function in script.js

Horizontal positioning: X values: negative = left of image, 0-100 = on image, >100 = right of image
